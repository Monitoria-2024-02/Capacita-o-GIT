- `git init` – inicia um repositório
- `git status` – mostra o estado atual
- `git add` – adiciona arquivos ao staging
- `git commit` – salva mudanças
- `git config` – configura nome/email
- `git help` – mostra ajuda

- `git clone <url>` - clona um repositório online
- `git checkout -b <branch_name>` - cria e entra em uma nova branch

- `git log` - historico de commits
- `git log --oneline --graph` - historico de commits bonito
- `gitk` - gerenciador do git
- `git reflog` - historico de ações
